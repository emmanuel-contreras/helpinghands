Introduction

How to Use the App

In this app you can be a requester, a helper or both! If you are a requester, you can create a task, list skills needed to complete that task and we will find helpers with skills to complete your task. As a helper, you can be suggested to peoples tasks or invite yourself to a task, if you get matched to a task, you can use your skills to complete the requesters task and earn some cash.

Profile

The profile page is used to let people know who you are and what your skills are. If you are a helper, the skills section is used to suggest tasks to you and match you with helpers.

Dashboard

The dashboard contains lists of tabs

  • My Tasks - This tab contains a list of your current active tasks.
  • Confirmed - This tab contains your requested tasks with helper matched.
  • Pending - This tab contains your tasks without a matched helper.
  • Suggested - This tab contains possible tasks which you would be able to complete as a helper.
  • Completed - This contains a history of tasks you have requested and have been completed.

Create Tasks

This page allows you to list a task on the app, helpers with the required skills will be matched to complete your task.

Settings

Basic app settings as well as some functionality can be found in this page.

Search

The search page allows you to search the database for specific users or tasks.

Note: You can always access this page under settings.
